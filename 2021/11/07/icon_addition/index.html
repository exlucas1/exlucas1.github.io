为hexo-Chic主题添加自定义icon

Lucas-Zh   November 7, 2021  15:44:33 Hexo  loading

前言

由于Chic中的样式基本为文字构成，为了做一些美化，且Chic提供的iconfont矢量图标较少，我想要寻找一种能够导入自己喜欢图标的方法。iconfont上有非常多精美的矢量图标，所以我将基于这个网站来实现我的目标。

具体操作

1.建立项目
找到你想添加的图标，将你想要添加的图标点击 “添加入库” 按钮。

2.添加至项目
点击右上角小购物车图标。这里会看到你刚刚添加的所有图标。

接下来就是为你的图标创建一个项目，名称随意。创建完成后你将看到如下页面。

3.生成在线链接
点击 Font class 按钮然后生成链接。

4.修改Chic主题配置文件
复制第3步生成的在线链接，找到themes/Chic/_partial/head.ejs，在该文件中找到，新增<link rel="stylesheet" href="//at.alicdn.com/t/font_2932977_r09dyogs5xa.css">这段代码，href中的内容即为生成的在线链接。自此，你已经可以根据适量图标的名称在你的博客中进行调用了。
