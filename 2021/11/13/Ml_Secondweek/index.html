Gradient Descent in Practice I - Feature Scaling#

Feature Scaling

Idea: Make sure features are on a similar scale

E.g. x1x_1x1​ = size (0-2000 feet2^22) transform to  size (feet 2 ) 2000\frac{\text { size (feet }^{2} \text { ) }}{2000}2000 size (feet 2 ) ​

​ x2x_2x2​ = number of bedrooms(1-5) transform to x2= number of bedrooms 5x_{2}=\frac{\text { number of bedrooms }}{5}x2​=5 number of bedrooms ​

Get every feature into approximately a −1≤xi≤1-1 \leq x_{i} \leq 1−1≤xi​≤1 range.

Mean normalization(均值归一化)

Replace xix_ixi​ with xix_ixi​ - μi\mu_{i}μi​ (average value of x in the training set) to make features have approximately zero mean(Do not apply to x0x_0x0​ = 1).

E.g. x1= size −10002000x_{1}=\frac{\text { size }-1000}{2000}x1​=2000 size −1000​

​ x2=#bedrooms−25x_{2}=\frac{\# b e d r o o m s-2}{5}x2​=5#bedrooms−2​

​ −0.5≤x1≤0.5,−0.5≤x2≤0.5-0.5 \leq x_{1} \leq 0.5,-0.5 \leq x_{2} \leq 0.5−0.5≤x1​≤0.5,−0.5≤x2​≤0.5
